# AI Portfolio Generator

AI Portfolio Generator is a web application that allows users to **create professional developer portfolios quickly and efficiently**. Users can fill in their personal and professional details in an easy-to-use interface, see a **live preview of the portfolio**, and **share or download it as a PDF**.

---

## Features

- **Left-side Editing Panel**: Input sections for personal information, education, skills, projects, achievements, and certificates.
- **Live Preview**: Real-time portfolio preview updates instantly as users edit details.
- **Shareable Link**: Generate a unique link to share the portfolio online.
- **Download as PDF**: Export the completed portfolio as a professional PDF.
- **Clean and Responsive UI**: Simple and modern interface for easy portfolio creation.

---

## Portfolio Sections

1. **Personal Information**
   - Full Name
   - Objective / Career Summary
   - Email, LinkedIn, GitHub

2. **Education**
   - Degree, College/University, CGPA, Year
   - Multiple entries supported

3. **Skills**
   - Add key technical skills or tools

4. **Projects**
   - Highlight project title, description, and technologies used

5. **Achievements**
   - Awards, recognitions, or notable accomplishments

6. **Certificates**
   - Add certifications with title and issuing organization

---

## Tech Stack

- **Frontend**: React, Next.js, Tailwind CSS
- **Backend & Database**: Firebase
- **AI Assistance**: Google Antigravity tools
- **Deployment**: Web-based, can be hosted on Firebase Hosting or Vercel

---

## How It Works

1. Users fill in their details on the **left editing panel**.
2. The **right panel shows a live preview** of the portfolio.
3. Users can **update details anytime** and see changes instantly.
4. Portfolio can be:
   - **Shared online** through a link
   - **Downloaded as PDF** for offline use
